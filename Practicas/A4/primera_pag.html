A4. Primeras páginas

En esta tarea, vamos a crear una página HTML5 válida y sencilla que muestre el uso de varias etiquetas. La página tiene que estar bien formada y el conjunto de caracteres será UTF-8.

El HTML debe pasar el siguiente validador : https://validator.w3.org/nu/

Escoge la opción File upload.

Si el validador de HTML te sugiere que agregues un atributo lang a la etiqueta html (es un warning, no un error) consulta la siguiente página para resolverlo: https://www.w3schools.com/tags/ref_language_codes.asp

Tu página DEBE CUMPLIR, además, los siguientes requisitos:

  1. Debe tener al menos un ejemplo de cada una de las siguientes etiquetas:
    • span
    • p
    • div
    • h1
    • ul
  2. Debe tener tres etiquetas a y tres etiquetas li.
  3. Debe tener texto en negrita y texto en cursiva, pero no puedes usar las etiquetas b o i.

En este enlace tienes un generador de texto automático que puede serte útil: https://lipsum.com/

Sube a la plataforma Moodle el enlace a tu repositorio GitHub y a la página web de GitHub.
